# Semify Enterprise API Documentation

This repository mirrors the Semify developer portal for AI-readable documentation and API integration resources.

## Source of truth

- Developer portal: https://developers.semify.com/
- OpenAPI spec (portal download): https://developers.semify.com/_spec/openapi.yaml?download

## Contents

- `openapi3_0.yaml`: OpenAPI 3.0 spec (kept in sync with the portal)
- `openapi.json`: OpenAPI spec in JSON format
- `api-docs-context7.md`: Context7-friendly summary and index
- `portal-markdown/`: Per-operation portal pages as Markdown
- `onboarding-workflow`: End-to-end onboarding guide (mirrors wiki content)
- `wiki/`: Snapshot of the GitHub wiki (starting with `Home.md`)

## Guidance

Use the developer portal for canonical human-readable docs and SDK generation. Use this repo for AI ingestion, indexing, and Context7 synchronization.
